## 📖 About the Portfolio

Welcome! This is my personal developer portfolio, designed to showcase my journey as a developer, academic milestones, core projects, and technical skills. Built on a modern React, TypeScript, and Tailwind CSS foundation, this site is designed with premium typography, harmonized dark/light themes, and expressive interactive elements to offer recruiters and developers an engaging visual experience.

---

## 🚀 Features

- **📱 Fully Responsive Design**: Mobile-first fluid layout optimized for immaculate cross-device accessibility and breakpoints.
- **🌗 Seamless Dark/Light Themes**: Dynamic ambient background glows and smooth transition colors tailored to user preferences.
- **✨ Smooth Micro-Animations**: Rich transitions and visual feedback driven by `motion` (Framer Motion).
- **📂 Interactive Projects & Credentials Showcase**: Modular tabs displaying projects, verification certificates (with PDF link options), and an interactive stack inspector.
- **🗺️ Interactive 3D Stack**: Responsive isometric interactive components highlighting backend, frontend, database, and system levels.
- **📬 Secure Contact Form**: Real-time validated messaging form integrated with Web3Forms API.

---

## 💻 Tech Stack

- **Core Framework**: React 19 (SPA Architecture)
- **Programming Language**: TypeScript (Strict type safety)
- **Styling & Theme**: Tailwind CSS & CSS Variables
- **Animations**: Motion (Framer Motion)
- **Build Tool**: Vite (Fast HMR & Optimized production bundler)
- **Icons**: Lucide React

## 🛠️ Installation and Setup

Follow these steps to run the portfolio project locally:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Clone the Repository
```bash
git clone https://github.com/its-sudharshan-45/portfolio.git
cd portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Set Up Environment Variables
Create a `.env` file in the root directory and add your [Web3Forms](https://web3forms.com/) Access Key to enable the contact form:
```env
VITE_WEB3FORMS_ACCESS_KEY=your_web3forms_access_key_here
```

### 4. Run the Development Server
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000`.

### 5. Build for Production
To generate a production bundle:
```bash
npm run build
```
